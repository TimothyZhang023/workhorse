# Workhorse 产品演进路线图

> 最后更新：2026-03-15  
> 状态：Phase 2 核心能力已完成，进入 Phase 3 准备阶段。

---

## 📅 演进历程与现状 (As-Is)

### ✅ 已完成 (v2.0 桌面优先时代)

- **核心架构转轨**: 从多租户 Web 应用切换为 `Tauri 2 + Node.js Sidecar` 架构。
- **单机身份管理**: 移除注册/登录流程，自动注入本地 `local` 用户，支持 SQLite 本地化存储。
- **完善的 Agent 引擎**: 实现基于 ReAct 循环的任务编排、Token 预算管理与长上下文压缩。
- **MCP 全力支持**: 完美支持 Stdio 与 SSE 两种 transport，内置 `shell_execute` 安全执行沙箱。
- **自动化闭环**: 集成 Cron Jobs 定时任务、任务历史审计 trace、钉钉 Webhook 触发。
- **多端点负载**: 支持 Endpoint 候选集与健康重试机制。
- **技能市场**: 提供默认技能模板库，支持 Git 仓库安装和 AI 自动生成。
- **MCP 市场**: 提供默认 MCP 模板库和社区市场集成，支持代码自动生成。
- **OpenAI 兼容**: 通过 `/v1/*` 代理接口，可作为其他 AI 工具的模型网关。

## 🚀 未来路线图

### 🟢 Phase 3: 核心提效与智能演进 (进行中)

-   **[ ] MCP Marketplace (Chat Style)**: 将插件市场重构为类似 GPT/Gemini 的对话引导形式，提供沉浸式工具发现体验。
-   **[ ] 启动更新检查**: 异步从 GitHub Release 检查新版本，确保用户实时获取核心引擎优化。
-   **[ ] agentEngine 单元测试**: 实现高密度的 Vitest 覆盖（>80%），确保 ReAct 循环与上下文压缩的稳定性。
-   **[ ] Skill 市场引导**: 调研并内置主流 Skill 提示词市场入口，提供一键下载/安装指南。
-   **[ ] 全方位性能优化**: 针对长对话内存占用、Sidecar 启动耗时进行专项调优。
-   **[ ] 安装包瘦身**: 优化渲染进程与 Sidecar 冗余依赖，提升分发效率。

### 🔵 Phase 4: 自主进化 (规划中)

-   **[ ] 任务主动提示**: 配合 Cron，实现早间新闻汇总、GitHub 监控报表自动推送。
-   **[ ] 小模型路由**: 简单任务自动路由至本地模型以节省成本并提升隐私。
-   **[ ] Artifacts 面板**: 完善代码、HTML 或 SVG 的侧边栏实时预览。
